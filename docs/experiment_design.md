# Experiment Design (A/B)

## Feature
**Partner Program Enrollment Banner** shown to eligible creators.

## Hypothesis
Showing an enrollment banner increases **Enroll Rate Given Eligible** without harming creator publishing behavior.

## Primary metric
Enroll Rate Given Eligible.

## Guardrails
- publish frequency (episodes/day)
- revenue per active creator

## Analysis
- diff in proportions with 95% CI
- segment checks (tier, country)
- interpret lift + practical impact
